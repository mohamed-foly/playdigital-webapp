## Make WebApp Steps

```bash

composer create-project --prefer-dist laravel/laravel webapp # create laravel app
php artisan serve # run command in seprated shell window to serve project
composer require laravel/ui # adding ui package
php artisan ui:auth # scaffolding authentication files
php artisan ui bootstrap # scaffolding bootstrap files
npm install # install ui dependencies
npm run dev # building ui
php artisan migrate # run migrations for database 

```
